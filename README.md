# 超市管理系统 java57

## 项目概述

本项目是基于B/S架构和MVC设计模式的超市管理系统，采用前后端分离的技术架构。系统提供完整的源码、数据库文件以及详尽的万字文档，旨在为超市提供一套高效、易用、稳定的解决方案。

## 技术栈

- **开发语言**：Java
- **前端框架**：Vue
- **后端框架**：SpringBoot
- **数据库**：Mysql
- **缓存**：Redis
- **开发工具**：IDEA
- **构建工具**：Maven
- **系统环境**：Windows
- **JDK版本**：1.8

## 功能模块

### 系统管理

- 角色管理
- 菜单管理

### 销售管理

- 销售主页
- 销售记录
- 积分兑换记录

### 人事管理

- 部门管理
- 员工管理

### 库存管理

- 仓库管理
- 出库明细
- 入库明细
- 供货商信息
- 库存统计

### 会员管理

- 会员信息管理

### 商品管理

- 分类管理
- 商品信息
- 积分商品
- 销售统计

## 核心亮点

- **前后端分离**：后端提供稳定API，前端负责界面展示，分离开发提高效率。
- **完善的权限管理**：可根据用户角色分配不同权限，确保系统使用安全。
- **详细的文档**：提供万字文档，详细说明系统使用和部署流程，易于上手和维护。
- **模块化设计**：功能模块划分清晰，便于扩展和维护。
